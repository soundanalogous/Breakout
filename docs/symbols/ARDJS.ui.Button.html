ARDJS.ui.Button(pin, buttonMode, longPressDelay)
An object to represent a physical button. The advantage of using the Button class over listening for pin change events on a Pin object, is that the Button class handles debouncing and provides helpful button events: Pressed, Released, Long Press and Sustained Press
Parameters:
{Pin} pin
A reference to the pin the button is connected to.
{number} buttonMode
The mode of the button (either PULL_DOWN or PULL_UP). Default is PULL_DOWN.
{number} longPressDelay
The delay time in milliseconds before a long press event is fired.
