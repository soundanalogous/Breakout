Compatible I/O Boards

The following is a list of I/O boards that have been successfully tested with Breakout.

Official Arduino Boards

  • Arduino Due, Zero, 101
  • Arduino Yun and Leonardo
  • Arduino Uno
  • Arduino Fio
  • Arduino Mega 2560 (all pins are usable)

Arduino-compatible boards

  • Teensy 2.0, Teensy++ 2.0
  • Teensy 3.0/3.1/3.2, Teensy LC

The following boards should work with Breakout, but have not yet been tested

  • Arduino Pro
  • Arduino Pro Mini
  • Arduino Mini (newer version)
  • Arduino Nano (newer version)
  • Arduino LilyPad (newer version)
  • Arduino Duemilanove
  • Arduino Diecimila
